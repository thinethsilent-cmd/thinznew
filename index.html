Binance Connection

Securely connect your Binance account. Your keys are encrypted. Never grant withdrawal permissions.

How to get API Keys

  1. Log into Binance and go to the API Management section from your profile menu.
  2. Create a New API: Choose 'System generated', enter a label like "Nexus Bot", and pass the security verification.
  3. Edit Restrictions: Click 'Edit restrictions'. You must check Enable Spot & Margin Trading so the bot can execute trades.
  4. Security Check: Leave Enable Withdrawals UNCHECKED. We will never ask for withdrawal access.
  5. Copy Keys: Copy both your API Key and Secret Key into the form above and save.
For detailed instructions, read Binance's official guide.
